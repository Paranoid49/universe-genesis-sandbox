# Universe Genesis Sandbox

一个由可组合宇宙公理驱动、确定性可复现且因果闭合的多本体宇宙演化沙盒。

玩家作为宇宙之外的提问者，创建初始条件并自由观察、实验、干预和比较宇宙。宇宙可以建立在不同本体与法则之上，但所有生成的“果”都必须能够追溯到对应的“因”。

## 项目定位

这不是严格的学术模拟器、传统策略游戏或事件浏览后台，而是一座让宇宙按照自身规则持续演化，并让玩家围绕自己选择的问题获取证据与验证理解的实验场。

核心体验：

- 选择、组合或生成宇宙公理与初始条件。
- 启动一个可以持续演化并确定性复现的宇宙。
- 自主选择观察对象、现象和问题，通过证据逐渐形成理解。
- 进行实验、施加干预或建立分支，并观察宇宙按照自身规则产生的后续变化。
- 回溯任意结果的原因，并把真实形成的历史沉淀为可分享和可创作的资产。

## 本地运行

安装依赖：

```text
npm install
```

启动开发服务器：

```text
npm run dev
```

在启动它的可见终端中按 `Ctrl + C` 可关闭开发服务器。

常用检查：

```text
npm test
npm run build
```

日常完整检查：

```text
npm run check
```

发布级检查还必须包含真实浏览器端到端测试：

```text
npm run check:release
```

## 当前进度

当前实现已完成原阶段 0 至阶段 8 的工程规格：

- 阶段 0：完成概念规格、MVP 边界、确定性原则和非目标约束。
- 阶段 1：完成可复现宇宙卡片 MVP，支持 seed 输入、随机 seed、分享码、8 至 12 条关键纪元事件和 10 个宇宙模板。
- 阶段 2：完成宇宙法则引擎，支持结构化法则、法则关系、指标影响来源和两个 seed 的法则差异对比。
- 阶段 3：完成宇宙时间线与纪元系统，支持 30 条以上事件、纪元筛选、事件因果解释和后续生成影响。
- 阶段 4：完成时间线影响摘要、局部对象生成偏置、星系/恒星系/行星/生物圈样本、局部探索浏览路径、阶段 5 文明候选种子、内容池分层、分享 current-only 版本策略和 CI 检查。
- 阶段 5：完成独立文明实体、文明路径、神话系统、8 至 15 条文明历史事件、文明浏览区域、阶段 6 禁区和自动化验收测试。
- 阶段 6：完成造物主干预规格、结构化奇迹、干预代价、干预日志、反噬状态、显式干预输入复现和阶段 7 禁区。
- 阶段 7：完成轻量二维宇宙观察台、宇宙/星系/恒星系三级投影、五类信息叠层、逐事件时间浏览、键盘节点操作和文字降级路径。
- 阶段 8：完成 A1 本地存档、收藏优先图书馆、组合搜索、确定性恢复、版本化 JSON 导入导出和存储失败原子性处理。

当前规则版本：`ugs-ruleset@0.7.0`，分享短码：`UGS070`，干预分享载荷版本为 `I1`。

阶段 7 只读消费既有结构化宇宙，阶段 8 只在 UI 基础设施层读写本地存档；二者都不改变规则版本或模拟核心的确定性结果。

上述阶段构成新的产品重构基线，但当前体验仍以结构化结果和事件浏览为主，不代表目标产品已经成立。后续实施以新的产品定义和产品重构路线为准。

产品重构步骤 1“可追因的宇宙生成器”至步骤 6“具有自主实体的世界”均已完成。步骤 6 首次独立评审发现的 6 项 P1 与 1 项 P2 已经过三轮有限复查全部关闭；步骤 7 尚未开始。
