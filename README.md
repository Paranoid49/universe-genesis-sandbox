# Universe Genesis Sandbox

一个 seed 驱动的宇宙创世沙盒。

用户播下一组初始法则，系统基于确定性随机过程生成一个自洽宇宙。这个宇宙可以接近现实物理，也可以包含魔法、神明、灵魂、奇迹、轮回、因果裂缝与文明飞升。

## 项目定位

这不是严格的学术宇宙模拟器，而是一个兴趣向、造物主视角、硬科幻与神话幻想混合的宇宙生成神话引擎。

核心体验：

- 输入或随机生成 seed。
- 生成一个可复现宇宙。
- 查看宇宙法则、时间线、代表性星系、行星生命样本、文明候选与神话事件。
- 通过分享码或链接复现同一个基础宇宙或带干预的宇宙分支。

## 本地运行

安装依赖：

```text
npm install
```

启动开发服务器：

```text
npm run dev
```

在启动它的可见终端中按 `Ctrl + C` 可关闭开发服务器。

常用检查：

```text
npm test
npm run build
```

完整检查：

```text
npm run check
```

## 当前进度

当前实现已完成阶段 0 至阶段 6：

- 阶段 0：完成概念规格、MVP 边界、确定性原则和非目标约束。
- 阶段 1：完成可复现宇宙卡片 MVP，支持 seed 输入、随机 seed、分享码、8 至 12 条关键纪元事件和 10 个宇宙模板。
- 阶段 2：完成宇宙法则引擎，支持结构化法则、法则关系、指标影响来源和两个 seed 的法则差异对比。
- 阶段 3：完成宇宙时间线与纪元系统，支持 30 条以上事件、纪元筛选、事件因果解释和后续生成影响。
- 阶段 4：完成时间线影响摘要、局部对象生成偏置、星系/恒星系/行星/生物圈样本、局部探索浏览路径、阶段 5 文明候选种子、内容池分层、分享 current-only 版本策略和 CI 检查。
- 阶段 5：完成独立文明实体、文明路径、神话系统、8 至 15 条文明历史事件、文明浏览区域、阶段 6 禁区和自动化验收测试。
- 阶段 6：完成造物主干预规格、结构化奇迹、干预代价、干预日志、反噬状态、显式干预输入复现和阶段 7 禁区。

当前规则版本：`ugs-ruleset@0.6.2`，分享短码：`UGS062`，干预分享载荷版本为 `I1`。

阶段 7 只能在 [阶段 7 准入检查](docs/phase-7-readiness.md) 与完整质量门禁通过后进入独立规格设计，不能反向破坏阶段 0 至阶段 6 的确定性结果。
