## 1. 可运行产品闭环

玩家进入默认运行主流程后，可以在“当前宇宙、观察、研究记录、已发生历史”之间切换。玩家选择观察方式获得有限信号，查看证据来源，将信号加入关注，记录笔记或推测，并在刷新后恢复研究记录继续探索。

当前提供四种互不依赖的观察方式：

- 结构观测：只暴露结构区间，不泄露精确凝聚值。
- 能量趋势：比较相邻逻辑时刻的方向，不宣称长期规律。
- 近期变化：区分已提交语义变化与当前范围内未观测到变化。
- 规律追踪：只显示已经参与实际状态转换的公开规则引用。

玩家可以从对象、尺度、时间或现象四类入口中的任意一种开始，入口最终形成包含对象、逻辑时刻和观察方式的显式请求。页面只持有冻结的观察访问端口、有限对象引用和公开公理，不接收完整 `UniverseState` 或运行控制器。

当前宇宙概览分别呈现当前可知状态和近期可观察变化。近期变化只来自玩家已经获得的变化信号；没有相关信号时只说明尚未观察，不把未知误报为没有变化。

产品不要求玩家寻找生命、文明、神明、异常或叙事高潮。空无、稳定、证据不足和确认不存在都是合法观察结果。

## 2. 契约与实现

### 2.1 四层边界

- `UniverseState` 继续作为运行事实唯一来源。
- `ObservableSignal` 和 `EvidenceRecord` 是玩家当前可获得的有限观察结果。
- `InternalKnowledgeClaim` 只建立步骤 6 的兼容边界，当前不生成占位内部信念。
- `PlayerFocus`、`PlayerNote` 和 `PlayerHypothesis` 是独立玩家研究数据，不进入宇宙事实或因果网络。

观察协议为 `ugs-observation@2`，证据协议为 `ugs-evidence@1`，研究记录协议为 `ugs-research-notebook@2`，研究存档协议为 `ugs-research-archive@2`，研究存储协议为 `ugs-research-storage@2`。

### 2.2 观察确定性与来源

观察身份绑定宇宙定义、被观察逻辑时刻对应的唯一状态、对象、观察方式、可见结果和证据引用。相同可信历史与显式观察请求产生相同信号、证据强度和稳定身份。

证据只引用实际状态或已提交转换。观察可以不完整、无结果或互相冲突，但不能创建底层不存在的事实。支持与反对证据同时存在时保留冲突状态，不静默覆盖任一来源。

包含观察信号的研究存档必须使用当前观察访问端口提供的可信校验器。校验器重新执行同一对象、时刻和方法的观察，核对精确状态身份、结论、知识状态、证据摘要、来源对象和因果节点；只重算内容哈希和外层校验和不能把同历史错误状态或其他伪造来源变成合法记录。

知识状态明确区分未观察、证据不足、已观察但无结果、部分支持、解释冲突、已确认与确认不存在。同一对象和观察方式形成稳定认知问题，支持、反对和中性证据分别保存，冲突结果经过存档往返后保持全部来源。

### 2.3 研究记录与存储

研究记录保存观察信号、关注、标签、笔记、推测和观察历史。存档校验以下不变量：

- 宇宙定义与当前运行历史身份匹配。
- 条目 ID 和顺序唯一且有效。
- 关注与观察历史引用已保存信号。
- 笔记和推测只引用已保存证据。
- 存档版本、外层身份和校验和有效。
- 每条信号都能由当前可信运行历史重新产生相同结论和来源。

浏览器使用独立数据库 `ugs.research.v2`。写入先完成校验与 IndexedDB 事务，再更新页面状态；写入拒绝或事务中止时保留先前有效版本、清除不再适用的成功提示、保留本次未保存输入并显示错误。

### 2.4 主流程与旧版隔离

默认主导航不再永久展示概览、星系、文明、奇迹、纪元、法则和日志。旧静态页面只在带有明确说明的 `LegacyApplication` 中按需装配，不作为运行、观察或研究事实来源。

新当前宇宙页不平铺底层对象数量和完整属性；新历史页只显示已提交变化，不暴露原始因果 ID 和未经观察的完整差异值。旧 `OverviewPage`、`LogsPage`、`ObservationConsole`、`SpaceExplorer`、`CivilizationPanel`、旧选择器和九页模型不得进入新主流程依赖图。

当前宇宙概览只消费观察信号、玩家关注、公开公理和未知观察方式。法则认知区分公开公理、观察支持的规律、内部观点边界和未知规则。传统空间条件视图只有在对应对象已有观察信号时出现，并且只展示可见对象引用和已获得信号，不读取完整对象属性。

## 3. 已保持的不变量

- 步骤 1 因果闭包、双向查询、规则来源、确定性随机证据和同果异因历史保持有效。
- 步骤 2 状态唯一事实来源、逻辑时钟、完整差异、运行因果、确定性恢复和存档语义保持有效。

- 旧 UGS070、I1、A1 和旧页面只在隔离兼容路径读取，不自动升级为研究记录或等价运行历史。
