## 使用规则

- 归档文档不得覆盖当前产品定义和实施路线。
- 新需求不得继续追加到归档阶段文档中。
- 归档文档仅在修复失效链接、补充归档说明或纠正历史事实时更新。
- 当前实现仍受旧契约影响时，应把有效约束写入当前架构、质量门禁或新的实施步骤，而不是重新启用旧阶段文档。
