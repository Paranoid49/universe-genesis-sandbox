- 空间统计、星系、恒星系、行星、生物圈、文明候选及对应负事实均登记追因入口。
- 时间线事件、当前事件、事件总数、影响摘要和局部偏置均登记追因入口。
- 当前法则领域、具体法则、法则关系、日志三类结果、文明统计与属性、神话、文明事件、奇迹统计、概率变化、指标变化、干预结果、目标变异与反噬均登记追因入口。

## 5. UI 运行时与体积

完整页面追因闭环使原 React 生产包超过 110 KiB 扩展保护线。生产构建与 UI 测试改为使用 Preact React-compat，统一别名位于 `ui-runtime.config.ts`，由 Vite 与 UI Vitest 配置共同消费。

兼容边界、测试要求和回退条件见 [步骤 1 用户界面运行时体积决策](../decisions/ui-runtime-size-step-1.md)。本次没有提高体积预算，也没有删除因果能力规避保护线。

迁移后发现 Preact 在 SVG 中把 `tabIndex` 渲染为大小写不兼容属性，导致观察节点无法获得真实浏览器焦点。组件已显式输出小写 `tabindex`，Chromium、Firefox、WebKit 和移动端 Chromium 均验证焦点样式与键盘路径恢复。
