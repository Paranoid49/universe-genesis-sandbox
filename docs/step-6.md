# 步骤 6：具有自主实体的世界

状态：已完成，首次独立评审发现的 6 项 P1 与 1 项 P2 经过三轮有限复查全部关闭。

本文件记录 [步骤 6 冻结验收口径](step-6-freeze.md) 的实际实现，不增加冻结范围之外的完成要求。步骤 7 的长期迁移、静态创作档案、结构化编年史和正式创作导出均未实现。

## 1. 用户获得的产品

- 宇宙可以完全没有主体，也可以在状态和宪法条件满足后形成自主实体。
- 主体基于有限感知、有限记忆和可能错误的信念形成意图，并通过统一规则执行器行动或被拒绝。
- 主体形成、行动、关系、公开叙述、失去自主性和消亡均进入已发生历史与因果网络。
- “自主实体”页面只在当前宇宙实际存在或曾经存在主体时出现，并只展示行为证据、生命周期、关系、公开叙述和行动因果。
- 宇宙事实、实体感知、实体信念、公开叙述、神话档案与真实主体使用不同契约，不因名称相同自动合并。

## 2. 协议与职责边界

- 自主协议为 `ugs-autonomy@2`，定义实体、感知、记忆、信念、意图、行动、关系、叙述、神话档案和转换记录。
- 宇宙宪法为 `ugs-constitution@3`，宪法模块为 `ugs-constitution-module@3`，规则执行器为 `ugs-rule-executor@3`。
- 宇宙定义为 `ugs-universe-definition@4`，运行状态为 `ugs-universe-state@6`，状态转换为 `ugs-state-transition@4`，运行存档为 `ugs-runtime-archive@6`。
- 运行因果网络为 `ugs-runtime-causality@6`，增加实体、感知、记忆、信念、意图、行动、关系、叙述和神话档案节点及对应边，并以所属状态转换区分内容相同但发生时刻不同的规则执行节点。
- 分支为 `ugs-branch@5`，分支存档为 `ugs-branch-archive@5`，创世条件包为 `ugs-genesis-package@4`，历史分支包为 `ugs-history-branch-package@5`。
- 运行存储为 `ugs-runtime-storage@5` 和 `ugs.runtime.v5`，分支存储为 `ugs-branch-storage@5` 和 `ugs.branches.v5`。

## 3. 运行数据流

1. `prepareAutonomyStep` 根据宪法策略和当前对象状态形成或停止主体，并产生本步感知、记忆、信念和意图。
2. 只有意图明确选择的 `trigger: "autonomous"` 规则才能进入 `executeConstitutionStep`。
3. 自主规则与环境规则共用条件、代价、约束、优先级、预算、随机决定和原子提交边界。
4. `completeAutonomyStep` 根据真实规则执行记录形成 applied、rejected 或 idle 行动，并更新行动记忆、关系和公开叙述。
5. 状态差异、自主转换和随机决定共同进入状态身份、存档、分支身份、历史哈希、事件投影和运行因果网络。

语义重放从已校验创世初态开始逐步重新执行转换，并逐项比较每个转换和最终事实。重放内部生成的中间状态不重复扫描此前全部历史，最终完整性校验仍覆盖完整状态和历史。

## 4. 认知与叙述边界

- 感知只能读取策略声明的当前对象字段，偏差和不确定性必须显式声明。
- 记忆具有宪法声明的有限容量和稳定淘汰顺序。
- 信念与宇宙事实分离，保存相信值、置信度、来源记忆和形成时刻。
- 意图只使用当前信念、记忆、实体状态和策略目标，不读取未来结果。
- 公开叙述只能引用实体实际形成的信念，不自动成为宇宙事实或真实神明。
- 页面不展示私有感知、私有记忆、完整信念、内部对象 ID、内部实体 ID或执行器拒绝枚举。

## 5. 完整性与威胁处理

`autonomy-validation.ts` 在恢复边界校验以下内容：

- 主体必须引用当前宇宙中的实际对象和宪法策略，策略目标类型必须匹配对象类型。
- 记忆来源必须对应实际感知或行动，信念、意图、行动、关系和叙述引用必须存在且属于同一主体。
- 活动主体不能携带停止时刻，停止主体必须具有合法生命周期，停止后不能继续行动。
- 主体、关系和叙述映射键必须与记录身份一致。
- 即使攻击者重新计算状态指纹和存档校验和，语义重放仍会拒绝被修改的认知历史和状态事实。

架构门禁禁止当前主流程依赖 `civilizationSeed`、`civilizations.ts`、`content/civilizations.ts`、`recipes/civilization.ts`、`CivilizationPanel` 和旧 `UniverseSummary`。旧文明与神话只允许在 `LegacyApplication` 隔离兼容视图读取。

## 6. 参考宪法与界面

- 新增 `living-tide@1`“潮生世界”参考宪法，初态包含两个非自主潮生体。
- 第一步只有环境历时增长，第二步形成两个主体并产生自主行动。
- 主体对活性能量具有确定性偏差，因此形成与宇宙事实不同但符合自身证据的信念。
- 行动消耗能量、增加信号、形成关系与公开叙述，能量耗尽后主体停止且关系终止。
- 非生物“觉醒回声”测试夹具使用同一自主协议，不存在生命或文明专用执行分支。
- 无主体参考宇宙不显示自主实体入口，也不创建文明、神话或主体占位页面。
